# Jujutsu (jj) Repo — No Git

This repo (`nomark`) uses **Jujutsu** (`jj`) as VCS, not Git.
Git is forbidden. Never stage, commit, push, branch, or otherwise use Git.
If asked for a commit message, provide message text only without running any VCS commands.
To read the working diff use `jj diff` or `git diff` (read-only).
